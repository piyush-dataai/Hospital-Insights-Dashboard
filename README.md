# 🏥 Hospital Insights Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive Hospital Insights Dashboard built in Microsoft Power BI. The dashboard provides a comprehensive view of hospital performance by analyzing patient records, admissions, departments, revenue, treatments, and operational KPIs.

The objective is to help hospital management make data-driven decisions through interactive visualizations.

---

# 🎯 Objectives

- Analyze patient admissions and discharges
- Monitor hospital revenue
- Understand department performance
- Track patient demographics
- Identify treatment trends
- Improve operational efficiency

---

# 🛠 Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Excel / CSV Dataset

---

# 📊 Dashboard Features

- Discharges
- Occupancy Rate

### Patient Analysis

- Admission Trends

### Operational Insights

- Bed Occupancy
- Average Length of Stay
- Department Utilization
